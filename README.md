Servicetechnische Website

In diesem Projekt sollten:
- Frontend Daten in einer Webpage eingegeben werden
- Diese Daten sollten in einer Datenbank gespeichert werden
- Aus diesen Daten sollte eine XML Datei erstellt werden
- Andere Mögliche XML Dateien sollten importiert werden und visualisiert werden

Datenbank die Benutzt werden sollte: MongoDB
